workWithopenMSE

Description

bamExtras is an R package to that contains functions and data to support stock assessments of fish stocks in the Southeast US Atlantic using the Beaufort Assessment Model (BAM).

A set of functions has been developed to use BAM assessment data and outputs to condition operating models that can be used with the openMSE package .

Example usage

Load the package

First, install and load the bamExtras package.

# Install and load package
#devtools::install_github("nikolaifish/bamExtras")
library(bamExtras)

# Load dependencies
library(stringr)

Work with the openMSE package

#install.packages("openMSE")
library(openMSE)

Work with results of BAM assessments

# see all data objects in bamExtras
# data(package="bamExtras")$result[,"Item"]

# standardize naming conventions and assign shorter name
rdat_RP <- standardize_rdat(rdat_RedPorgy)

Build Stock and Fleet objects

Stock_RP <- rdat_to_Stock(rdat_RP)
Fleet_RP <- rdat_to_Fleet(rdat_RP)

Assemble operating model

OM_RP <- new("OM", Stock=Stock_RP, Fleet=Fleet_RP, Obs=Precise_Unbiased, Imp=Perfect_Imp)

Run basic MSE

MSE_RP <- runMSE(OM_RP)
